# Project creation

1. Install VUE cli.
```shell
$ npm install -g @vue/cli
```

2. Create proyect with VUE cli.
```shell
$ vue create vue-frontend
```

3. ```cd``` to the project directory and start serving.
```shell
$ cd vue-frontend
$ npm run serve
```
